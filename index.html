Moderator commands explained - intro:
When you are the first person to join a room, you automatically become the moderator of that room. Use commands like /kick, /mute, and /mod to moderate your room!
Written on Mar 13, 2013, 1:45 PM
Moderator commands explained - kick:
/kick <user> <reason> kicks a user from your room (back to the lobby). You must specify a username, but a reason for kicking that user is optional. (The user kicked will see the reason for being kicked).
Written on Mar 13, 2013, 1:45 PM
Moderator commands explained - mute:
/mute <user> will prevent a user currently in your room from sending any public messages until unmuted by a moderator. Muted users can still send private messages.
Written on Mar 13, 2013, 1:45 PM
Moderator commands explained - unmute:
/unmute <user> will unmute a user currently muted, allowing that user to send public messages again.
Written on Mar 13, 2013, 1:45 PM
Moderator commands explained - mod:
/mod <user> will make that user account a moderator account of the same room you are in.
